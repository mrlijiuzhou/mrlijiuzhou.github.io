关于 nodeJs版本的 一些坑

下载了 hexo 和 hexo cil工具 ,因为项目需要变更 node 版本导致吧node版本切换为8.版本,后来发现找不到hexo,hexo相关命令也启动不了,重下载也报错
后来才知道是因为Node版本问题
采用nvm管理版本

1.查看安装的版本及正在使用的版本
nvm ls
2.使用某一版本
nvm use ${版本号}
